== Description ==

#### Featured Image Caption

The Simplest: Featured Image Caption adds a caption field right below featured image so you don't need to go to library media to write captions for images. It is also fully compatible with Gutenberg and Classic Editor. Unlike other alternatives, this plugin is up-to-date!

== Installation ==

To install this Plugin please follow:

* Please install the plugin in your Wordpress Dashboard under Plugins
* Activate the plugin
* Either check a checkbox in settings to print captions automatically or use `<?php echo get_ts_fic_caption( $post_id ); ?>` in your code to print it manually.
